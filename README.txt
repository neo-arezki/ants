ANTs v0.3.0 — Journal mensuel intelligent
==========================================

INSTALLATION / MISE À JOUR

1. Fermez ANTs.
2. Décompressez ce dossier où vous voulez.
3. Double-cliquez sur lancer_ants.bat.

La base reste dans :
C:\Users\VOTRE_NOM\Documents\ANTs\ants.db

Au premier lancement, ANTs :
- sauvegarde automatiquement la base existante dans Documents\ANTs\backups ;
- migre la base v0.2 vers v0.3 ;
- conserve toutes les écritures, projets et comptes d'épargne.

NOUVEAUTÉS

- journal présenté mois par mois ;
- navigation avec les flèches gauche/droite ;
- une écriture créée depuis un ancien mois reçoit automatiquement une date de ce mois ;
- auto-complétion des libellés déjà saisis ;
- apprentissage du type, du poste et du caractère récurrent d'un libellé ;
- nouveau type « Mouvement » pour les retraits d'espèces et virements internes ;
- sous-postes de transport : carburant, train, transports urbains, parking/péage ;
- reclassement prudent de certains anciens libellés connus ;
- modification par double-clic ;
- total mensuel des revenus, dépenses et mouvements ;
- gestion des comptes d'épargne.

IMPORTANT

Un retrait d'espèces est à saisir comme :
Type : Mouvement
Poste : Mouvements › Retrait d'espèces

Il ne sera donc pas compté comme une dépense, mais ANTs pourra suivre votre besoin mensuel de cash.
