# SPECIFICATIONS ANTs

## Principes
ANTs est local, descriptif et non culpabilisant. Les données utilisateur sont prioritaires sur le code.

## Écritures
Une écriture possède une date, un type, un montant, un libellé, un poste, un caractère habituel ou ponctuel, un statut confirmé ou prévisionnel, un projet facultatif et un commentaire facultatif.

## Prévisionnel
Une écriture future peut être prévisionnelle. Son montant reste modifiable. Sa date peut rester à préciser. Une écriture habituelle n'a pas nécessairement un montant fixe.

## Projets
Une dépense peut être associée à un projet de vie. Un projet constitue un dossier comprenant une description générale et la liste de ses achats associés.

ANTs distingue obligatoirement :
- les dépenses confirmées, déjà réalisées ;
- les dépenses prévisionnelles, encore à venir ;
- l'engagement total, somme des deux ;
- le reste prévisionnel après épargne, dépenses confirmées et prévisions.

Le commentaire d'une écriture permet d'expliquer un achat particulier sans alourdir la description générale du projet.

## Lecture visuelle
La couleur indique le type d'écriture. L'icône indique le poste. Le statut prévisionnel est signalé séparément.
