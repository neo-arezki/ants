# SPECIFICATIONS - ANTs

## Vision

ANTs est un assistant personnel de gestion financière.

Il n'a pas vocation à remplacer un logiciel de comptabilité ou une banque.

Son objectif est d'aider son utilisateur à :

- comprendre ses habitudes financières ;
- préparer ses projets ;
- suivre son budget simplement ;
- améliorer progressivement sa situation financière.

Le logiciel est entièrement local.

Les données de l'utilisateur sont plus importantes que le code.

---

# Principes

## Une écriture = un événement financier

Chaque écriture possède :

- une date
- un type
- un montant
- un libellé
- une catégorie
- un commentaire (optionnel)

---

## Types d'écriture

Trois types existent.

### Revenu

Exemple :

- salaire
- prime
- remboursement

---

### Dépense

Exemple :

- courses
- carburant
- restaurant

---

### Mouvement

Un mouvement ne constitue pas une dépense.

Il déplace simplement l'argent.

Exemples :

- retrait DAB
- dépôt espèces
- virement vers Livret A
- retour du Livret A

---

# Les catégories

Les catégories servent uniquement à analyser les dépenses.

Elles ne doivent pas compliquer la saisie.

Les libellés sont prioritaires.

---

# Les libellés

Le libellé est l'élément principal d'une écriture.

ANTs apprend automatiquement :

- le commerçant
- la catégorie
- le caractère récurrent
- (plus tard) le montant moyen

L'utilisateur doit pouvoir saisir une écriture en quelques secondes.

---

# Journal

Le journal constitue l'écran principal.

Les écritures sont regroupées par mois.

Le mois affiché devient automatiquement le mois des nouvelles écritures.

---

# Tableau de bord

Le tableau de bord présente notamment :

- revenus
- dépenses
- mouvements
- épargne
- charges fixes
- projets

---

# Coach

Le coach ne juge jamais.

Il observe.

Il explique.

Il propose.

Il ne culpabilise pas l'utilisateur.

---

# Philosophie graphique

L'interface doit être :

- moderne
- lisible
- rapide

Les couleurs servent uniquement à améliorer la compréhension.

Les icônes facilitent la lecture.

L'information doit rester prioritaire sur la décoration.
